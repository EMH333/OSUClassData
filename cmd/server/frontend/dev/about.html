OSU Class Data Explorer - Created by Ethan Hampton

This is a web application that allows you to explore the data provided by mirrorkeydev/osu-grades acquired through a FOIA request for Oregon State University classes. I wrote a brief blog post on how to go about FOIA requests. The included data is between Fall 2018 and Fall 2021, and only includes classes with 10 or more students. Hopefully this will help your class forecasting in some way!
