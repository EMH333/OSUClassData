OSU Class Data Explorer - Created by Ethan Hampton

This is a web application that allows you to explore the data provided by _______ acquired through a FOIA request for Oregon State University classes. The included data is between Fall 2019 and Summer 2021. Hopefully this will help your class forecasting in some way!
